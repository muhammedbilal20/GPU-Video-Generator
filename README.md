# GPU Video Generator

A browser-based GPU Benchmark Video Generator built with HTML, CSS and JavaScript.

## Features

- Add unlimited GPUs
- Enter FPS or Relative Performance
- Automatic sorting
- 1080 × 1920 Canvas
- Animated benchmark
- Background video
- Background music
- WebM recording
- GitHub Pages compatible

## How to Use

1. Open the website.
2. Enter GPU names.
3. Enter FPS values.
4. Click **Generate**.
5. Click **Record**.
6. The video downloads automatically.
